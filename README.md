# BigData-Labs

Repository contenente il codice delle esercitazioni relative al corso di [BigData](torlone.dia.uniroma3.it/bigdata/) tenuto nell'anno accademico 2020/2021. Le esercitazioni riguardano:
- Hadoop (3.2.2)
- MapReduce
- Apache Hive (2.3.8)
- Spark (3.1.1)
